Easy Customization Guide

Customize Any Page in Under 5 Minutes

Every page uses CSS custom properties (variables). Change the values at the top and all colors update instantly.

1

Change Brand Colors

Open any HTML file → find :root { } at the top of <style> → change --primary, --secondary, --accent to your hex colors.

2

Update Headline & Copy

All text is plain HTML. Find the <h1>, <h2>, <p> tags and edit directly. No frameworks, no components — just text.

3

Change Prices

Search for the price amount (e.g., 297) and replace with your price. Crossed-out prices and "save $X" copy update accordingly.

4

Connect Pages

Each CTA button links to the next page via href="...". Update the href to your own URLs or payment processor links.

5

Add Countdown Timer

Find TIMER_HOURS in the JavaScript at the bottom of checkout/upsell pages. Change the number to set how many hours until expiry.

6

Replace Testimonials

Each testimonial card is a self-contained <div class="testimonial-card">. Edit the quote text, name, title, and the initials inside the avatar circle.
